# app-menu-en-c
# Sistema de Domicilios en C
Aplicación de consola en C que simula un pedido a domicilio con menú interactivo.
# Qué hace

Muestra un catálogo de 6 productos
Permite agregar productos al carrito
Calcula el total con IVA (19%) y domicilio ($5.000)
Confirma el pedido y vacía el carrito

# Estructuras de control usadas

do…while — menú principal
for — recorrer catálogo y carrito
if / else — validaciones de entrada
